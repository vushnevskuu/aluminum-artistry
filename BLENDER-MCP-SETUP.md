# Blender MCP — готово к работе

## Что уже сделано

- uv установлен/обновлён
- blender-mcp установлен (через uvx)
- Addon `blender-mcp-addon.py` скачан в папку проекта
- MCP‑сервер настроен в `.cursor/mcp.json`

## Что сделать в Blender

1. Откройте Blender
2. Меню **Edit → Preferences → Add-ons**
3. Нажмите **Install...** и выберите `blender-mcp-addon.py` (в корне этого проекта)
4. Включите аддон **Interface: Blender MCP**
5. В боковой панели 3D View нажмите **N** и откройте вкладку **BlenderMCP**
6. Нажмите **Connect to Claude**

## Перезапуск Cursor

Перезапустите Cursor, чтобы MCP‑сервер подключился к Blender.

После этого можно использовать Blender через AI в Cursor.
